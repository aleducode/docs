## Prerrequisitos

<Check>Docker y Docker Compose instalados</Check>
<Check>Git instalado</Check>
<Check>Al menos 4GB de RAM disponible</Check>

## Instalación

### 1. Clonar el repositorio

```bash
git clone https://github.com/aleducode/conveypro.git
cd conveypro
```

### 2. Configurar variables de entorno

```bash
cp .envs/.local/.django.example .envs/.local/.django
cp .envs/.local/.postgres.example .envs/.local/.postgres
```

Editar `.envs/.local/.django`:

```env
DJANGO_DEBUG=True
DJANGO_SECRET_KEY=tu-secret-key-aqui
DJANGO_SETTINGS_MODULE=config.settings.local
DATABASE_URL=postgres://postgres:postgres@postgres:5432/convey
REDIS_URL=redis://redis:6379/0
```

### 3. Levantar servicios

```bash
docker-compose -f local.yml up -d
```

<Info>
La primera vez tardará unos minutos mientras descarga las imágenes.
</Info>

### 4. Aplicar migraciones y crear superusuario

```bash
docker-compose -f local.yml run --rm django python manage.py migrate
docker-compose -f local.yml run --rm django python manage.py createsuperuser
```

### 5. Acceder a la aplicación

| Servicio | URL |
|----------|-----|
| **App Principal** | http://localhost:8000 |
| **Admin Django** | http://localhost:8000/admin |
| **Mailpit** | http://localhost:8025 |
| **Flower** | http://localhost:5555 |

## Comandos Útiles

<AccordionGroup>
  <Accordion title="Ver logs">
    ```bash
    docker-compose -f local.yml logs -f django
    docker-compose -f local.yml logs -f celeryworker
    ```
  </Accordion>

  <Accordion title="Shell de Django">
    ```bash
    docker-compose -f local.yml run --rm django python manage.py shell
    ```
  </Accordion>

  <Accordion title="Migraciones">
    ```bash
    docker-compose -f local.yml run --rm django python manage.py makemigrations
    docker-compose -f local.yml run --rm django python manage.py migrate
    ```
  </Accordion>
</AccordionGroup>
