## Uso en Tailwind

### Colores de Texto

```html
<!-- Textos principales -->
<p class="text-gray-900">Texto principal</p>
<p class="text-gray-600">Texto secundario</p>
<p class="text-gray-400">Texto deshabilitado</p>

<!-- Estados -->
<p class="text-green-600">Éxito</p>
<p class="text-red-600">Error</p>
<p class="text-yellow-600">Advertencia</p>
<p class="text-blue-600">Información</p>
```

### Colores de Fondo

```html
<!-- Fondos de página -->
<div class="bg-gray-50">Fondo de página</div>
<div class="bg-white">Fondo de card</div>

<!-- Fondos de estado -->
<div class="bg-green-50">Fondo éxito</div>
<div class="bg-red-50">Fondo error</div>
<div class="bg-yellow-50">Fondo advertencia</div>
<div class="bg-blue-50">Fondo información</div>
```

## WhatsApp Preview

Colores para la previsualización de mensajes estilo WhatsApp:

```css
/* Fondo del chat */
.wa-chat-bg {
    background-color: #e5ddd5;
    background-image: url('/static/img/wa-pattern.png');
}

/* Burbuja mensaje entrante */
.wa-bubble-incoming {
    background-color: #ffffff;
}

/* Burbuja mensaje saliente */
.wa-bubble-outgoing {
    background-color: #dcf8c6;
}

/* Check de leído */
.wa-check-read {
    color: #53bdeb;
}
```

## Variables CSS Personalizadas

Si necesitas definir variables globales:

```css
/* static/css/custom.css */
:root {
    --color-primary: #0f172a;
    --color-secondary: #1e293b;
    --color-accent: #3b82f6;

    --sidebar-width: 16rem;
    --header-height: 4rem;

    --border-radius: 0.5rem;
    --shadow-sm: 0 1px 2px 0 rgb(0 0 0 / 0.05);
    --shadow-md: 0 4px 6px -1px rgb(0 0 0 / 0.1);
}
```
