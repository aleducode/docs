## Botones

### Primario

```html
<button class="bg-slate-900 text-white px-4 py-2 rounded-lg hover:bg-slate-800 transition-colors">
    Guardar
</button>
```

### Secundario

```html
<button class="bg-white text-gray-700 px-4 py-2 rounded-lg border border-gray-300 hover:bg-gray-50 transition-colors">
    Cancelar
</button>
```

### Danger

```html
<button class="bg-red-600 text-white px-4 py-2 rounded-lg hover:bg-red-700 transition-colors">
    Eliminar
</button>
```

### Con Icono

```html
<button class="flex items-center gap-2 bg-slate-900 text-white px-4 py-2 rounded-lg">
    <svg class="w-5 h-5"><!-- Heroicon --></svg>
    Nueva Campaña
</button>
```

### Loading State

```html
<button class="bg-slate-900 text-white px-4 py-2 rounded-lg opacity-75 cursor-not-allowed" disabled>
    <svg class="animate-spin h-5 w-5 mr-2 inline"><!-- Spinner --></svg>
    Procesando...
</button>
```

## Modales

```html
<!-- Alpine.js modal -->
<div x-data="{ open: false }">
    <button @click="open = true" class="btn-primary">Abrir Modal</button>

    <div
        x-show="open"
        x-cloak
        class="fixed inset-0 z-50 flex items-center justify-center"
    >
        <!-- Overlay -->
        <div
            @click="open = false"
            class="fixed inset-0 bg-black/50"
        ></div>

        <!-- Modal -->
        <div class="relative bg-white rounded-xl shadow-xl max-w-md w-full mx-4 p-6">
            <h3 class="text-lg font-semibold text-gray-900">Título del Modal</h3>
            <p class="mt-2 text-gray-600">Contenido del modal...</p>

            <div class="mt-6 flex justify-end gap-3">
                <button @click="open = false" class="btn-secondary">
                    Cancelar
                </button>
                <button class="btn-primary">
                    Confirmar
                </button>
            </div>
        </div>
    </div>
</div>
```

## Dropzone (Upload)

```html
<div
    class="border-2 border-dashed border-gray-300 rounded-xl p-8 text-center hover:border-blue-400 transition-colors cursor-pointer"
    x-data="{ dragging: false }"
    @dragover.prevent="dragging = true"
    @dragleave="dragging = false"
    @drop.prevent="dragging = false"
    :class="{ 'border-blue-400 bg-blue-50': dragging }"
>
    <svg class="w-12 h-12 mx-auto text-gray-400"><!-- Upload icon --></svg>
    <p class="mt-4 text-gray-600">
        Arrastra tu archivo aquí o
        <span class="text-blue-600 font-medium">selecciona</span>
    </p>
    <p class="mt-1 text-sm text-gray-400">Excel (.xlsx) hasta 10MB</p>
    <input type="file" class="hidden" accept=".xlsx,.xls">
</div>
```
