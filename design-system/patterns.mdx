## Wizard Pattern

Patrón para flujos multi-paso:

```html
{% extends "base.html" %}

{% block content %}
<div class="max-w-3xl mx-auto">
    <!-- Progress steps -->
    <div class="flex items-center justify-center mb-8">
        {% for step in steps %}
        <div class="flex items-center">
            <div class="flex items-center justify-center w-8 h-8 rounded-full
                {% if step.completed %}bg-green-500 text-white
                {% elif step.current %}bg-blue-500 text-white
                {% else %}bg-gray-200 text-gray-500{% endif %}">
                {% if step.completed %}
                    <svg class="w-5 h-5"><!-- Check --></svg>
                {% else %}
                    {{ step.number }}
                {% endif %}
            </div>
            <span class="ml-2 text-sm {% if step.current %}font-medium text-gray-900{% else %}text-gray-500{% endif %}">
                {{ step.name }}
            </span>
            {% if not forloop.last %}
            <div class="w-16 h-0.5 mx-4 {% if step.completed %}bg-green-500{% else %}bg-gray-200{% endif %}"></div>
            {% endif %}
        </div>
        {% endfor %}
    </div>

    <!-- Step content -->
    <div class="bg-white rounded-xl shadow-sm border border-gray-100 p-6">
        <h2 class="text-xl font-semibold text-gray-900 mb-4">
            {{ current_step.title }}
        </h2>

        {% block step_content %}{% endblock %}

        <!-- Navigation -->
        <div class="flex justify-between mt-8 pt-6 border-t border-gray-100">
            {% if not is_first_step %}
            <a href="{{ prev_url }}" class="btn-secondary">
                Anterior
            </a>
            {% else %}
            <div></div>
            {% endif %}

            <button type="submit" class="btn-primary">
                {% if is_last_step %}Finalizar{% else %}Siguiente{% endif %}
            </button>
        </div>
    </div>
</div>
{% endblock %}
```
