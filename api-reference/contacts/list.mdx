## Endpoint

```
GET /api/contacts/
```

## Parámetros de Query

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `search` | string | Buscar por nombre, teléfono o email |
| `whatsapp_line` | integer | Filtrar por línea WhatsApp |
| `has_conversation` | boolean | Solo contactos con conversaciones |
| `page` | integer | Número de página |
| `page_size` | integer | Resultados por página (default: 20) |

## Ejemplo de Uso

<CodeGroup>

```bash cURL
curl -X GET "https://app.convey.pro/api/contacts/?search=juan" \
  -H "Authorization: Token your-token-here"
```

```python Python
import requests

response = requests.get(
    "https://app.convey.pro/api/contacts/",
    params={"search": "juan", "page_size": 100},
    headers={"Authorization": "Token your-token-here"}
)

contacts = response.json()
print(f"Encontrados: {contacts['count']} contactos")
```

</CodeGroup>

## Errores

| Código | Descripción |
|--------|-------------|
| 401 | Token no proporcionado o inválido |
| 403 | Sin permisos para ver contactos |
