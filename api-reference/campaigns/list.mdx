## Endpoint

```
GET /api/campaigns/
```

## Parámetros de Query

| Parámetro | Tipo | Descripción |
|-----------|------|-------------|
| `status` | string | Filtrar por status |
| `whatsapp_line` | integer | ID de la línea WhatsApp |
| `template` | integer | ID del template |
| `created_after` | datetime | Creadas después de esta fecha |
| `created_before` | datetime | Creadas antes de esta fecha |
| `page` | integer | Número de página |
| `page_size` | integer | Resultados por página |

### Status disponibles

| Status | Descripción |
|--------|-------------|
| `DRAFT` | Borrador, no iniciada |
| `SCHEDULED` | Programada para envío futuro |
| `PROCESSING` | En proceso de envío |
| `PAUSED` | Pausada manualmente |
| `COMPLETED` | Finalizada |
| `CANCELLED` | Cancelada |

## Ejemplo de Uso

<CodeGroup>

```bash cURL
curl -X GET "https://app.convey.pro/api/campaigns/?status=PROCESSING" \
  -H "Authorization: Token your-token-here"
```

```python Python
import requests

response = requests.get(
    "https://app.convey.pro/api/campaigns/",
    params={"status": "COMPLETED"},
    headers={"Authorization": "Token your-token-here"}
)

campaigns = response.json()
for campaign in campaigns["results"]:
    rate = campaign["delivered_count"] / campaign["sent_count"] * 100
    print(f"{campaign['name']}: {rate:.1f}% entregados")
```

</CodeGroup>

## Errores

| Código | Descripción |
|--------|-------------|
| 401 | Token no proporcionado o inválido |
| 403 | Sin permisos para ver campañas |
